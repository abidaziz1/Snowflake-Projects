# Projects

This directory contains individual Snowflake projects. Each project should have its own subdirectory with:

- A descriptive README.md
- SQL scripts for setup and execution
- Any configuration files or data samples

## Example Project Structure

```
project-name/
├── README.md           # Project description and instructions
├── setup.sql          # Database, schema, and table creation
├── data/              # Sample data files (if applicable)
├── queries/           # Analytics queries
└── README.md          # Project documentation
```
